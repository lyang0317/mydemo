关系型数据库服务 RDS

关系型数据库服务 (Relational Database Service) RDS旨在为您提供廉价、安全、可靠的关系型数据库服务，您可以将您的数据库系统搭建在此。

新旧方法对比

如果您在业务中需要使用数据库，那么。有新旧两种方法可以选择：

旧方法
  • 第1步：买或者租一台服务器
  • 第2步：买一个外网IP（可省略）
  • 第3步：搭建数据库（或者数据库集群）
  • 第4步：做负载均衡
  • 第5步：搭建安全、备份环境
  • 第6步：分配/管理数据库账号以及权限
  • 第7步：使用JDBC/客户端等连接使用
总时间投入：约 xxx 天
总 体 金 额：约 xxx 元
新方法
  • 第1步：注册数之云账号
  • 第2步：1秒开通RDS服务
  • 第3步：可视化快速新建项目空间和数据库
  • 第4步：可视化快速分配账号
  • 第5步：使用JDBC/客户端等连接使用，
  • 其中，权限管理和数据安全都是自动完成。
总时间投入：约 5 分钟
总 体 金 额：约 动态调整，按需收费

功能

专业的数据库集群
RDS使用Mysql Cluster 来搭建数据库集群，更 容易扩展，且保证了高 可靠性
强大的数据迁移工具
RDS自带数据迁移工具， 不管您原来的数据库在 哪里，都能轻松迁移到 数之云RDS上
完善的大数据平台工具
RDS背后有一整套我们 的大数据平台支撑工具。 更大可能的为您的数据 存储和分析着想
专业的负载均衡
我们的集群实现多机负 载与同时查询，更大可 能地为您优化操作速度

优势

数据安全
RDS提供给您更安全的 数据支持，我们给您默 认提供双机热备功能， 如果机房出现什么问 题，我们都将最大努力 地保证您的正常使用， 可用性达99.999%

简单易用
兼容Mysql，使用JDBC 驱动，数据迁移更加人 性化
